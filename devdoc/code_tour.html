    Tour of the Source Code¶

    Following is a specially tailored log output from Cyclus that is intended to give a high level understanding of the cyclus internals and code flow of execution. Each log statement has a link that, when clicked, shows related snippets of code from the cyclus core or related modules. The code snippets are intended to give developers a feel for the type of code they will need to write in order to get things done using Cyclus.
