Cymetric API¶

The following modules make up the cymetric package. Note that most of the important stuff you can get directly from cymetric without needing to import the module. For example:

from cymetric import metric

Interface Modules:

  • cymetric.schemas - Schemas
  • cymetric.evaluator - Evaluators
  • cymetric.execution - Execution
  • cymetric.filters - Filters
  • cymetric.graphs - Graphs

Metrics Modules:

  • cymetric.metrics - Metrics
  • cymetric.root_metrics - Root Metrics
  • cymetric.cycamore_root_metrics - Cycamore Root Metrics
  • cymetric.brightlite_root_metrics - Bright-lite Root Metrics
  • cymetric.fco_metrics - Fuel Cycle Option Metrics

Helper Modules:

  • cymetric.main - Command Line Interface
  • cymetric.tools - Tools
  • cymetric.timeseries - Timeseries
