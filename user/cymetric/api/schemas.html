cymetric.schemas - Schemas¶

Schemas for cymetric.

class cymetric.schemas.SchemaProperty(obj, prop)¶

Represents specific properties of a schema.

cymetric.schemas.canon_column(col)¶

Returns canonical form of a column.

cymetric.schemas.canon_dbtype(dbtype)¶

Returns the canonical form for a Cyclus database type.

cymetric.schemas.canon_name(name)¶

Returns canonical form of a column name.

cymetric.schemas.canon_shape(shape, dbtype)¶

Returns the canonical shape for a Cyclus database type.

class cymetric.schemas.schema(x)¶
